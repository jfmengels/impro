# Triangle amoureux

Nombre de joueurs : 3.
Tags : .

## Description

### Préparation

Chaque joueur va dans sa tête choisir un joueur qu'il aime ou apprécie beaucoup et un joueur qu'il déteste.

### Cours du jeu

Les joueurs jouent une scène normale, mais doivent toujours tous se trouver sur scène.
